This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

Marketing SPA (**Info no Zap**, infonozap.com.br) for vehicle plate and CPF consultation services via WhatsApp. Routes: `/` (hub), `/placas`, `/cpf`, and respective `/planos` pages. Frontend-only — no backend, no API, no database.

## Quick Reference

- **Build:** `bun run build`
- **Dev:** `bun run dev` (port 8080)
- **Lint:** `bun run lint`
